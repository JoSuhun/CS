# OSI 7계층

## 2계층에서 하는일
- 하나의 네트워크 대역(같은 네트워크 상)에서 존재하는 여러 장비들 중에서, 어떤 장비가 어떤 장비에게 보내는 데이터를 전달함
- 오류제어, 흐름제어

### 2계층 네트워크 크기
- 2계층은 하나의 네트워크 대역 LAN에서만 통신 가능함
- 다른 네트워크와 통신할 때는, 항상 3계층이 도와줘야 함
